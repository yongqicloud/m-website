# Project-Liyongqi-M-Website

#### 介绍
{**猫耳FM**}
一个二次元的FM网站

#### 软件架构
RMVC
    r:Router
    m:Modules
    v:views
    c:Controllers
#### 技术要点
**项目**
    gulp工程化管理工具
    gitee 代码托管
    gulp-sass && node-sass编译工具
    http-proxy-middleware 中间件接口代理工具
    webpack-stream 打包工具
    art-template 模板引擎
    moment 日期处理插件
    ...
**页面布局**
    `适用于市场上现有的95%以上的移动端`
    rem布局
    100%布局
    yo框架
**逻辑层**
    SPA(single page application) 单页面应用
    路由中转:整个项目只有一个入口文件，即index.html;
